# BLE RADAR OMEGA — MANUAL

## Mode simple
Utiliser COMMANDER Center pour :
- startup check
- briefing condensé
- workflow conseillé
- manuel local

## Centres
- NEXUS: historique, timeline, persistance
- OMEGA-X: base de connaissance, labels, rapport quotidien
- ARGUS: priorités, case files
- SENTINEL: escalades, campaigns, watch sessions
- ATLAS: co-présence, clusters, groupes
- HELIOS: vue exécutive
- AEGIS: incidents composés, playbooks
- ORACLE: projection de risque
- NEBULA: synthèse maître, dossiers
- CITADEL: check-up, export, maintenance

## Workflow recommandé
1. Scan
2. HELIOS
3. ARGUS
4. SENTINEL
5. AEGIS
6. ORACLE
7. NEBULA
8. CITADEL

## En cas d’alerte forte
1. Vérifier watchlist
2. Ouvrir ARGUS case file
3. Vérifier SENTINEL escalations
4. Exporter incident / audit
5. Sauvegarder watch session
6. Générer incident pack CITADEL
